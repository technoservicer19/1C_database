1С:Предприятие 7.7 Типовая конфигурация "Торговля+Склад", редакция 9.0
Релиз 7.70.906 от 26.02.2002

ВНИМАНИЕ! Переход на редакцию 9.0 конфигурации "Торговля+Склад" с 
редакции 8.7 невозможен с помощью объединения этих конфигураций. Ведение
учета с использованием новой редакции 9.0 возможно только
в новой информационной базе.
Для перехода на новую редакцию можно использовать пустую информационную
базу, созданную автоматически при установке редакции 9.0. 
В новую базу можно перенести данные из конфигурации редакции 8.7.
Последовательность действий при переносе данных из конфигурации 
"Торговля+Склад" редакции 8.7 в редакцию 9.0 подробно описана в файле 
CV87_90.txt, находящемся в каталоге ExtForms новой информационной базы 
(редакции 9.0).
При выполнении конвертации убедитесь, что вы используете последнюю версию 
внешней компоненты v7plus.dll. Для обновления компоненты скопируйте файл 
v7plus.dll из каталога информационной базы с конфигурацией редакции 9.0, 
созданного данным обновлением в каталог исходной информационной базы 
(с конфигурацией редакции 8.7).

Порядок обновления релиза 7.70.905 на релиз 7.70.906
========================================================================

Для обновления релиза конфигурации следует использовать режим
"Объединение конфигураций". Вам потребуется файл метаданных (файл 1Cv7.MD)
нового релиза конфигурации. 
Релиз конфигурации для обновления установлен в
подкаталоге UPDATE. После обновления подкаталог UPDATE может быть удален 
или использован для следующих обновлений.

1. Запустите систему 1С:Предприятие в режиме "Конфигуратор".
2. Сделайте архивную копию вашей информационной базы.
3. Вызовите режим "Объединение конфигураций", для этого
   в меню "Конфигурация" выберите пункт "Объединение конфигураций".
4. В запросе "Открыть файл конфигурации" выберите 
   файл метаданных 1Cv7.MD нового релиза конфигурации.
5. В окне режима "Объединение конфигураций" проверьте:
   - в группе "Приоритет конфигурации" был установлен 
     переключатель "Загружаемая конфигурация";
   - в группе "Метод объединения" установлен 
     переключатель "Замещать объекты".
6. Нажмите кнопку "OK" для объединения конфигураций.
7. После завершения будет открыто окно "Конфигурация",
   содержащее конфигурацию с внесенными изменениями.
   Выполните сохранение конфигурации.
8. По окончании установки обновления новые или измененные 
   внешние обработки, необходимые для работы Интернет-поддержки, 
   и файл схемы обмена информацией в электронном виде в формате XML,
   располагаются в каталоге UPDATE\ExtForms. Данные файлы не входят
   в конфигурацию, поэтому для того, чтобы их обновить, содержимое данного
   каталога следует перенести в каталог ExtForms вашей информационной базы.
9. Для работы с Интернет-поддержкой и с обменом данными в формате XML 
   убедитесь, что вы используете последнюю версию внешней компоненты 
   v7plus.dll. Для обновления компоненты скопируйте файл v7plus.dll
   из каталога UPDATE в каталог вашей информационной базы.
10.Внешние обработки для формирования печатных форм документов находятся в
   каталоге UPDATE\ExtForms\PrnForms. Данные файлы не входят
   в конфигурацию, поэтому для того, чтобы их обновить, содержимое данного
   каталога следует перенести в каталог ExtForms\PrnForms вашей 
   информационной базы.
11.Внешние обработки для работы с торговым оборудованием находятся в
   каталоге UPDATE\ExtForms\Equip. Данные файлы не входят
   в конфигурацию, поэтому для того, чтобы их обновить, содержимое данного
   каталога следует перенести в каталог ExtForms\Equip вашей 
   информационной базы.
12.Если необходимо вести учет в валюте ЕВРО, то следует добавить в 
   справочник валют соответствующий элемент 
   (код - 978, наименование - EUR), предварительно переписав файл прописей 
   EUR.spl в каталог информационной базы.
